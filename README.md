# adu-s-music-player
A sleek, responsive web music player featuring a dark-themed UI, shuffle/repeat modes, and dynamic control

Adu's Player is a clean, modern, and responsive web-based music application designed for a focused listening experience. Built with HTML, CSS, and JavaScript, it features a professional dark-themed interface with violet accents.

## Key Features
* **Intuitive Controls**: Play, pause, skip, and rewind functionality.
* **Advanced Playback**: Integrated shuffle and repeat modes for personalized listening.
* **Dynamic Progress Tracking**: Real-time track duration display with interactive seeking.
* **Volume Management**: Built-in slider for precise audio level control.
* **Curated Playlist**: Includes a pre-configured library of diverse tracks.

## Project Structure
* **index.html**: Contains the structure and layout of the player.
* **style.css**: Manages the dark-themed styling, responsive design, and UI components.
* **script.js**: Handles core application logic, including audio playback, event listeners, and track state management.

## Getting Started
To run this project locally, ensure your audio files are placed in an `assets/` folder to match the path configuration, then open `index.html` in any modern web browser.
